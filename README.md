# 방치형 RPG 게임
(게임 이미지 임시로 텍스트로 대체)
---
## 🎮게임 소개
- 게임을 켜놓기만 하면 플레이어 캐릭터가 자동으로 좀비를 잡음
- 일정 수의 좀비를 잡으면 날이 바뀜
- 10일차마다 추가 좀비 등장 (최대 4마리까지 추가 등장)
- 좀비를 잡으면 좀비의 종류 마다 다른 게임 속 재화가 들어옴
- 벌은 재화로 플레이어의 스탯을 강화 하거나 회복, 부스트 아이템을 구매 할 수 있음


---

## 느낀 점
- 제출 이틀 전 부터 인터넷이 끊겼다 들어왔다를 반복하다 결국 제출 이틀 전 새벽부터 제출 날 까지 인터넷이 끊겨 해결 + 필요한 내용을 서칭하기 어려운 상황이 되어 생각만큼 구현을 하지 못 하여서 아쉬웠음.
리드미 또한 원래 기획은 게임 소개 및 기능을 각 기능마다 gif도 같이 첨부 할려고 하였는데 시간이 부족해져서 위와 같이 소개에 모든 기능을 다 적기로 하게 되었다.
